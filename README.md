### Start with dev server
```
npm start
```

### Build production
```
npm run build:prod
```

### Build development
```
npm run build:dev
```

### Run eslint check
```
npm run lint:ts
```

### Run eslint fix
```
npm run lint:ts:fix
```

### Run stylelint check
```
npm run lint:scss
```

### Run stylelint fix
```
npm run lint:scss:fix
```
